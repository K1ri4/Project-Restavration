О проекте

Project Restavration — это инновационный проект, направленный на создание полнодуплексной связи между двумя электронно-вычислительными машинами (ЭВМ) с использованием нестандартного подхода. В основе решения лежит передача данных через бытовую электросеть 220В с применением доступных компонентов:

  • USB-TTL преобразователя CH340 - для обеспечения связи между компьютером и аппаратной частью;
  • Сгоревшей платы Arduino UNO - в качестве USB-TTL преобразователя;
  • Удлинителя 220В - в качестве физической среды передачи сигнала.

Цель проекта

Демонстрация возможности организации стабильной двусторонней передачи данных через бытовую электросеть с минимальными затратами, используя подручные компоненты. Проект исследует устойчивость связи, влияние помех и потенциальные применения в условиях, где стандартные методы передачи данных недоступны.

Особенности реализации

  • Аппаратная часть построена на базе неисправной Arduino UNO, что позволяет сократить стоимость эксперимента.
  • USB-TTL преобразователь CH340 обеспечивает совместимость с большинством современных компьютеров.
  • Электросеть 220В используется как альтернатива специализированным линиям связи (например, Ethernet over Powerline).

Потенциальные применения

  • Аварийная связь в условиях повреждённой сетевой инфраструктуры.
  • Обмен данными между устройствами в удалённых локациях без использования Wi-Fi или проводного интернета.
  • Образовательные цели — изучение основ цифровой связи и работы с электросетью как средой передачи.

Project Restavration сочетает в себе элементы реверс-инжиниринга, апсайклинга электроники и экспериментов в области передачи данных, предлагая нестандартный взгляд на привычные технологии.

Используемые технологии

  • C++ - frontend программы (UI/UX)
  • HTML и CSS — сайт и README.
  • Python (pyserial) - реализация USB-TTL протокола.
  • GitHub — контроль версий и синхронизация данных.
